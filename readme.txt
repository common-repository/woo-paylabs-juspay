== Installation ==

1. Upload the plugin files to the '/wp-content/plugins/woo-paylabs-juspay' directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the JUSPAY WooCommerce payment gateway -> Settings screen to configure the plugin
